## 📊 关键参数对照表

### Base面 判断条件
```
dot_up = normal · up_axis

is_base_face  ⟺  dot_up > 0.65
不是base  ⟺  dot_up ≤ 0.65

物理意义: base面几乎垂直向上
范围: cos(θ) > 0.65 → θ ∈ [0°, ~49°]
```

### Side面 判断条件
```
abs_dot_up = abs(normal · up_axis)

is_side_face  ⟺  abs_dot_up < 0.45
不是side  ⟺  abs_dot_up ≥ 0.45

物理意义: side面基本水平
范围: cos(θ) < 0.45 → θ ∈ [~63°, 90°]
```

### 遮挡检查
```
point_in_solid = false

test_point = xyz + normal * 0.1
if is_inside_any_other_solid(test_point):
    → face is covered (遮挡) → skip
else:
    → face is visible (可见) → include
```

### 距离优先级
```
distance < tol * 20  必须满足
distance最小 > distance较小 > distance较大

选择离点最近的面
```

## 🧮 数学计算步骤

### Step 1: 投影点到面
```
输入: point P, surface S
算法: GeomAPI_ProjectPointOnSurf(P, S)
输出: 投影点Q, 参数(u,v), 距离d
```

### Step 2: 计算法向量
```
输入: 投影点Q在参数(u,v)处
算法: GeomLProp_SLProps.Normal()
输出: 未归一化的法向量 N
```

### Step 3: 处理方向
```
if face.Orientation == REVERSED:
    N = -N
else:
    保持 N 不变

输出: 已处理方向的法向量
```

### Step 4: 归一化
```
N_unit = N / ||N||
输出: 单位法向量 [nx, ny, nz]
```

### Step 5: 合并法向量
```
selected = [n1, n2, n3]
sum_vec = n1 + n2 + n3
pose_normal = sum_vec / ||sum_vec||
输出: 最终的单位向量
```

## 🎬 完整计算流程 (伪代码)

```python
def compute_all_point_vectors(step_file, final_json, output_json):
    # 加载数据
    shape = load_step_file(step_file)
    solids = extract_all_solids(shape)
    solid_infos = classify_solids(solids)
    
    points = load_final_json(final_json)["points"]
    
    # 初始化
    classifiers = [BRepClass3d_SolidClassifier(s) for s in solids]
    summary = {regular:0, hole_arc:0, breakpoint:0, 
               non_base:0, fallback:0, failed:0}
    
    # 遍历每个点
    for point_id, point in points.items():
        xyz = point["xyz"]
        role = point["role"]
        
        # Step 1: 收集候选法向量
        candidates = face_normals_at_point(shape, xyz, tol)
        
        # Step 2: 去重 + 排序
        unique = collect_unique_normals(
            solids, xyz, incident_solid_ids, tol, classifiers
        )
        
        # Step 3: 检查特殊情况
        if on_through_hole_arc(xyz, hole_arcs):
            if is_non_base_point(xyz, solid_infos):
                selected = select_hole_non_base_plane_normals(...)
                method = "hole_arc_non_base_..."
            else:
                selected = select_hole_base_weld_direction_normals(...)
                method = "hole_arc_base_..."
            summary["hole_arc"] += 1
        
        elif role == "breakpoint":
            selected = select_contact_face_normals_for_solids(...)
            method = "breakpoint_..."
            summary["breakpoint"] += 1
        
        elif is_non_base_point(xyz, solid_infos):
            selected = select_non_hole_non_base_three_normals(...)
            method = "regular_non_base_..."
            summary["non_base"] += 1
        
        else:  # 普通点
            selected = select_regular_normals(unique, up_axis)
            method = "regular_three_contact_faces"
            summary["regular"] += 1
        
        # Step 4: 不足3个时降级
        if len(selected) < 3:
            fallback = select_lower_level_fallback_face(...)
            if fallback:
                selected.append(fallback)
                method += f"_with_{fallback['role']}_fallback"
                summary["fallback"] += 1
        
        # Step 5: 合并 + 保存
        pose_normal = combine_normals(selected)
        if pose_normal is None:
            summary["failed"] += 1
            continue
        
        point["pose_normal"] = pose_normal
        point["pose_normal_method"] = method
    
    # 保存结果
    save_json(output_json, {"points": points})
    print(f"Summary: {summary}")
```
